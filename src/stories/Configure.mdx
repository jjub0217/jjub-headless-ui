# jjub-headless-ui

웹 접근성이 내장되고 스타일 커스터마이징이 가능한 Headless UI 컴포넌트 라이브러리입니다.

## Headless UI란?

로직과 접근성만 제공하고, 스타일은 포함하지 않는 컴포넌트입니다.
`className` 또는 `data-state` 속성으로 자유롭게 스타일을 적용할 수 있습니다.

- **스타일 없음** — 색상, 폰트, 간격이 포함되어 있지 않음
- **접근성 내장** — WAI-ARIA 역할, 키보드 네비게이션, 포커스 관리
- **자유로운 스타일링** — CSS, Tailwind, CSS-in-JS 등 어떤 방식이든 사용 가능
- **의존성 없음** — React만 peer dependency
- **TypeScript 지원** — 전체 타입 정의 포함

## 설치

```bash
npm install jjub-headless-ui
```

## 빠른 시작

```tsx
import { Tabs } from 'jjub-headless-ui'

function App() {
  return (
    <Tabs.Root defaultValue="tab1">
      <Tabs.List>
        <Tabs.Tab value="tab1">탭 1</Tabs.Tab>
        <Tabs.Tab value="tab2">탭 2</Tabs.Tab>
      </Tabs.List>
      <Tabs.Panel value="tab1">첫 번째 탭 내용</Tabs.Panel>
      <Tabs.Panel value="tab2">두 번째 탭 내용</Tabs.Panel>
    </Tabs.Root>
  )
}
```

## data-state로 스타일링

모든 컴포넌트는 `data-state` 속성을 제공합니다:

```css
/* 활성 탭 */
[role="tab"][data-state="active"] {
  font-weight: bold;
}

/* 비활성 탭 */
[role="tab"][data-state="inactive"] {
  color: gray;
}
```
